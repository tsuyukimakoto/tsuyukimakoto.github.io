    オレには難しすぎるぜ

    nazoさんが謎のつぶやき をしていたので…

    ruby 1.8.6 (2008-03-03 patchlevel 114) [universal-darwin9.0]

        >> x = (1 == 0) or 2
    => 2
    >> y = (1 == 0) || 2
    => 2
    >> x
    => false
    >> y
    => 2

    ここまではいいんだが

        >> z = ((1 == 0) or 2)
    => 2
    >> z
    => 2

    おばかなオレには難しすぎる。 演算子

    やっぱオレは ||さえも無い Pythonでいいや。

    Python 2.5.1 (r251:54863, Jan 13 2009, 10:26:13)

        >>> x = (1 == 0) or 2
    >>> x
    2
    >>> y = ((1 == 0) or 2)
    >>> y
    2
