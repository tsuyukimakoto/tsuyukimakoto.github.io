    TextMateからMercurialを使う

    Mercurial勉強会に向けて、とりあえず触る丂！

    Mercurial自体はeasy_installで入った。2009/03/04に1.2が出たらしいけど、まだ1.1.2だった。まぁ、気にしない。

      $ easy_install Mercurial

    Mercurial自体の使い方は、 入門Mercurial に任せるとして、

    TextMateからMercurialが使いたいよ！

    Bundleがあった。ので、入れる。

        $ mkdir -p ~/Library/Application\ Support/TextMate/Bundles
    $ cd ~/Library/Application\ Support/TextMate/Bundles
    $ svn co http://svn.textmate.org/trunk/Bundles/Mercurial.tmbundle

    そしたら、TextMateで Bundles → Bundle Editor → Reload Bundles とかしてみる。面倒だったらTextMateを再起動する。そもそも起動してなかったら起動するだけ。

    これで、TextMateからMercurialが使えるようになった。

    • ctl + shift + M キーを押すとメニューが出る。

    • Log -v とか選ぶとこんな感じに表示される。

    • コミットだってできちゃう（当然か）。
