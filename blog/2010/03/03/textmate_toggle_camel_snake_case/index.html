    TextMateでCamelCaseとsnake_caseを変換する

    すぐに忘れるのでメモ。 Bundles → Bundle Editor → Edit CommandsとかやってBundle Editorを開く。

    SourceのBundleからToggle camelCase / snake_case / PascalCaseを選択して、Activationを無難なものに変更するだけ。

    option+^ くらいが使いやすいに違いない。変更したら、Reload Bundlesしたら、あとは変換したいやつを選択して、設定したキーを押すだけ。
