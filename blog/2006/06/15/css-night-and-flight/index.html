    CSS NiteとFlightにいってきた

    CSS Nite と CSS Flight に行ってきました。

    CSSを学ぼうという気と、デザイナーを見つけられたらいいなという気で同僚と行ってみました。

    残念ながら目的は達成できませんでした。

    大いに盛況で、時間に到着したときには181番目。アップルストア銀座4Fでストリーミングでの聴講となってしまいました。

    結局ストリーミングがうまくいかなかったり、たちっぱつらかったり、技術屋にはちょっと・・・な内容だったりで、CSS Niteは途中で失礼しました。

    その後のCSS Flightではいくつかのミニプレゼンを座って聴講することが出来ました。が、いかんせんmixiのコミュニティがベース＆テーブルが4人がけということもあり、知らない人とは殆ど会話することもなく終わってしまいました。日々の努力が足りないんでけど、だってmixi面倒じゃん。。

    謎が沢山、「MTとPHPを組み合わせると強力！」とか「MT3.3ではコードがI18N化！」とか。おいおい・・・。SixApartに入りたいという技術屋さんが謎。とか言ってみる。

    近々業外のサービスを立ち上げる予定なんだけど、結局デザイナー見つからずだなぁ。
