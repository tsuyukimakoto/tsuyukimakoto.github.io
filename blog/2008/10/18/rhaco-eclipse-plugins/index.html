    rhacoのEclipseプラグイン

    yabekenさんが Rhaco Development Tools というrhaco用のEclipsePluginsを開発しているようです（すでに動作する）。

    そもそもrhacoは、Eclipseで各種補完ができることを思想の中心においています。

    が、今激しく開発中のrhaco2はマジックメソッド？とかいうものを使うことになっていて、補完が難しくなっていました。なんでも、クラスに変数を定義すると暗黙でアクセッサが動的に存在するようになるとか（これで、C#やPythonのproperty的考えが使える）。

    そこで登場したのが rhaco 用 Eclipse Plugins の Rhaco Development Tools。Rhaco Development Tools を使うと、マジックメソッドを補完してくれるようになるそうです（ yabekenさんエントリ の画面ダンプ参照）。いままで、補完を気にして実装されてこなかった動的系のアイデアが沢山あったので、このRhaco Development Toolsの登場によりrhacoの変態度が増していくことでしょう。

    もう一つの思想、「どこでも動く」を守るために、PHP自体のカスタマイズや独自エクステンションの導入というPHP自体を変える方向性がとれないため、このEclipse Pluginsの登場はrhacoにとってかなり大きなインパクトです。

    ついでに、Rhaco::importに渡す文字列も補完ができるようになってます。詳しくは、 yabekenさんのエントリ で。

    ちなみに、 rhaco-users.jp からRhaco Development Toolsのダウンロードを行うには、 rhaco-users.jp の rhaco をダウンロードする というリンクをクリックして、rhaco関連物のダウンロードページへ行けばよいようです。
